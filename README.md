# Programa Desenvolve 2024 do Grupo Boticário em parceria com Alura

## Modelagem Conceitual, Lógica e Física de Banco de dados
- Este repositório consiste em arquivos com exercícios desenvolvidos em aulas do curso da Alura ('Modelagem de banco de dados: entidades, relacionamentos e atributos' e 'Modelagem de banco de dados relacional: modelagem lógica e física')

## Etapas:
### Modelagem de banco de dados: entidades, relacionamentos e atributos
- Identificar um modelo lógico e um modelo físico de dados
- Diferenciar um modelo de alto nível de um modelo de baixo nível
- Aplicar técnicas de modelagem de dados
- Diferenciar o Modelo Entidade e Relacionamento (MER) do Diagrama Entidade e Relacionamento (DER)
- Elaborar um modelo de banco de dados no brModelo

### Modelagem de banco de dados relacional: modelagem lógica e física
- Aprender a diferençar dos tipos de modelagem de banco de dados
- Conhecer a modelagem lógica e física de dados
- Exercitar a utilização do Power Architect
- Criar tabelas, colunas e representações de cardinalidade no modelo físico
- Produzir a representação das chaves primárias e estrangeiras
